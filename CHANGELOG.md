## [0.1.0] - 2026-04-22

### Added
- Initial public release of the `chat-me` monorepo.
- Self-hosted chat backend API (`apps/api`) with PostgreSQL persistence.
- Operator admin console (`apps/admin`) for conversation workflows.
- Embeddable widget bundle (`apps/widget`) and reusable SDK (`packages/sdk`).
- Public repository documentation, contribution guide, CI workflow, and community templates.

### Notes
- This release establishes the first public baseline for development and external contributions.
